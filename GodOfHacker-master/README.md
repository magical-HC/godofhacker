- 挂黑页、DDOS、社管理员全家、日后台、挂马、断室友网

- 百度、谷歌批量日站、免费上网、破解各种网吧

- 盗QQ号、刷Q币、刷会员、刷心悦3、日运营商免费打电话充话费

- 进内网、翻墙、澳门赌场、IP定位、顺着网线砍人

- 日教务系统、挖洞、发布漏洞、自动代码审计、读内存

- 根据文件一键破解、逆源码
